About Me

I am a Linguist/ESL teacher turned developer and a San Diego native. While living and working in China teaching English to Chinese and Korean university students, I discovered Linux, figured out how to use the command line, and taught myself bash scripting.

Following my return to San Diego I studied Java, Ruby on Rails, and Javascript. I have experience building apps using Ruby on Rails, designing APIs with Spring Boot Java, and building and deploying Node.js serverless APIs using AWS Lambda and API Gateway.

HidIt

Chrome plugin written with jQuery.

HidIt is a Google Chrome Extension written in Javascript/jQuery that allows the user to turn virtually any web page into an interactive flash card. Simply click a word to cause it to disappear. Click it again to cause the word to reappear. If you need a little help, hover the mouse over the whitespace to see the word in a different color.
